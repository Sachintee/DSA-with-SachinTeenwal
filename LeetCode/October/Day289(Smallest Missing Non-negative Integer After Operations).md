# 🚀 _Day 289. Smallest Missing Non-negative Integer After Operations_ 🧠


The problem can be found at the following link: [Problem Link](https://leetcode.com/problems/smallest-missing-non-negative-integer-after-operations)

## 🎯 **My Approach:**


## Code(C++)
```cpp
class Solution {
public:
    int findSmallestInteger(vector<int>& nums, int value) {
        int cnt[value];
        memset(cnt, 0, sizeof(cnt));
        for (int x : nums) {
            ++cnt[(x % value + value) % value];
        }
        for (int i = 0;; ++i) {
            if (cnt[i % value]-- == 0) {
                return i;
            }
        }
    }
};
```

## Code (Python)

```python
class Solution:
    def findSmallestInteger(self, nums: List[int], value: int) -> int:
        cnt = Counter(x % value for x in nums)
        for i in range(len(nums) + 1):
            if cnt[i % value] == 0:
                return i
            cnt[i % value] -= 1
```
